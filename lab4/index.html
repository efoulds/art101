Outline for making art:

  1. Determine what form of art to create
  2. Gather supplies
  3. Brainstorm ideas
  4. Outline or sketch concept
  5. Finalize concept
  6. Execute idea
  7. Add finishing touches

    8. Description: high level outline of how to make art

Outline for rock paper scissors game:

  1. Restart the game before each turn
  2. Accept user input
  3. Display three user options (rock, paper, scissors)
  4. Validate user move
  5. Computer randomly chooses one of three options
  6. Win or draw condition

    7. Description: high level outline of how to make a basic computer rock paper scissors game
